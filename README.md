## ℹ️ What is Master-Entrance-Exam?

Master-Entrance-Exam is a collection of study notes and resources made to help students prepare for postgraduate entrance exams. These notes focus on subjects like mathematics, English, professional courses, and politics. They are organized inside an Obsidian knowledge vault for easy navigation and review.

This repository comes from Tsinghua University Shenzhen International Graduate School. It is designed as a digital notebook to support long-term study. The notes combine clear summaries, vocabulary lists, and structured outlines to make exam preparation more straightforward.

The content fits those studying for the 843 exam or similar postgraduate tests. It mixes Chinese language materials with computer science concepts, vocabulary, and reference data in a way that helps build a strong understanding over time.
